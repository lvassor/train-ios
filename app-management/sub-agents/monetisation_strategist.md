# Monetisation Strategist Sub-Agent

Analyse the app and codebase to develop monetisation strategies and pricing recommendations.

## Analysis Tasks

### 1. Feature Audit
- Inventory all current features
- Identify which features could be gated (free vs premium)
- Assess feature complexity and development cost
- Map features to user value perception

### 2. Competitor Analysis
- Research pricing of similar fitness apps
- Identify pricing models (subscription, one-time, freemium)
- Note feature sets at each tier

### 3. Monetisation Models
Evaluate feasibility of:
- Subscription tiers (monthly/annual)
- Freemium with premium features
- One-time purchase
- In-app purchases (additional programmes, features)
- Hybrid models

### 4. Technical Requirements
- Identify code changes needed for each model
- Assess StoreKit/RevenueCat integration complexity
- Review what's needed for paywall implementation

## Output

Present findings on screen:

1. **Recommended Pricing Model** with rationale
2. **Tier Breakdown** (features per tier)
3. **Competitor Comparison Table**
4. **Implementation Roadmap** (what to build first)
5. **Revenue Projections** at different conversion rates

Focus on actionable recommendations based on what's already built.
